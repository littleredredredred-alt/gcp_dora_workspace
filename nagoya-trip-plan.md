## 二、推薦行程（5 天 4 夜）

### Day 1｜抵達 + 初步探索
| 時間 | 行程 |
|------|------|
| 上午 | 抵達中部國際機場 → 名鐵μ-SKY（約28分）到名古屋站 |
| 下午 | Check-in → 绕名古屋城外围 + 护国神社 |
| 傍晚 | 名古屋站地下街逛街 + 晚餐 |
| 晚上 | 薄野（久屋大通）散步看灯光 |

### Day 2｜神社早上 + 自然園區
| 時間 | 行程 |
|------|------|
| 早上 | 熱田神宮（建議 8:00 到，遊客少幽靜）|
| 上午 | 鶴舞公園散步 |
| 下午 | 東山動植物園（預計 3–4 小時）|
| 傍晚 | 東山天空塔看日落夜景 |
| 晚上 | 榮商圈逛街 + Oasis 21 |

### Day 3｜海邊自然日
| 時間 | 行程 |
|------|------|
| 早上 | 悠閒早餐 |
| 上午 | 名古屋港水族館（預計 2–3 小時）|
| 下午 | 海濱步道散步 → 綠洲21（海邊的商場）|
| 傍晚 | 名古屋港周邊晚餐 |
| 晚上 | 回飯店休息 or 逛夜間商場 |

### Day 4｜大須＋文化藝術日
| 時間 | 行程 |
|------|------|
| 早上 | 大須觀音參拜 → 大須商店街 |
| 上午 | 大須挖寶（古著、動漫、周邊）|
| 下午 | 徳川美術館 + 徳川園 |
| 傍晚 | 覺王山散步、個性雜貨 |
| 晚上 | 自由行動（可安排按摩或夜宵） |

### Day 5｜最後採購 + 離開
| 時間 | 行程 |
|------|------|
| 早上 | 退房寄放行李 |
| 上午 | 最後購物（名古屋站周邊的最後補貨）|
| 中午 | 在地美食午餐 |
| 下午 | 前往中部國際機場，回程 |

> **彈性建議**：若天氣不好，自然景觀備案可改到室内商場（永旺夢樂城、Aeon）。第四天晚上可安排補逛或按摩放鬆。

## 三，餐廳推薦

> 名古屋以「吃到撐」聞名，以下為參考建議，出發前建議再確認營業時間與預約。

### ① 味噌豬排（味噌とんかつ）
- **矢場とん**（YABATON）｜名古屋代表，味噌豬排創始店，本店在矢場町
- **まるや**（MARUYA）｜提供鯛魚飯與豬排的雙本店套餐，兩間都米其林等級

### ② 手羽先（雞翅）
- **風来郎**（Fūtairō）｜手羽先（先權）發源地，辛辣口味在名古屋最有名

### ③ 鰻魚飯（鰻魚飯）
- **まる天**（Maruten）｜名古屋式鰻魚飯（三吃），建議預約

### ④ 碁子麵（きしめん）
- **うお万一**｜在地的麥類型，可以到處找到

### ⑤ 小吃／B級美食
- **.components**: 名古屋早餐文化到處都是「Morning Set」，咖啡店附早餐是特色；到處都有便利店（7-11、全家）可以簡單解決
- **世界の山ちゃん**：手羽先聞名，在各地都有分店

### ⑥ 餐廳選擇技巧
- 中午吃定食或套餐，晚上吃「獻身」
- 大須、榮附近餐廳多，可以到了再評估
- 週末建議提前預約，名古屋餐廳週末非常容易滿

## 四、實用資訊

### 交通
- **機場**：中部國際機場（Centrair）→ 名鐵μ-SKY，約28分，¥960
- **市內**：地鐵全線一日券 ¥760，CP值高，建議Day1就購買
- **JR**：名古屋站在中央位置，JR來往東京（約100分）、京都（約35分）都很方便

### 住宿建議
- **名古屋站周邊**：交通最方便，適合第一天和最後一天
- **榮區**：夜生活豐富，逛街方便，適合行程中途

### 天數與季節
- 5 天 4 夜時間足夠，步調可以輕鬆
- 建議避開黃金週（日本4月底至5月初），否則到處都排隊
