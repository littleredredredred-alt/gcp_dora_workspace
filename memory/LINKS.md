# LINKS

收藏覺得值得留存的連結。

## Entry Format

```markdown
- YYYY-MM-DD HH:MM UTC | type: link | source: {telegram|discord|webchat}
  title: {連結標題}
  url: {URL}
  why_it_matters: {為什麼要收藏}
  related: {comma-separated tags}
```

## Usage

- 當使用者說「記一下這個連結」、「保存連結」、「收藏」時寫入
- 搭配 semantic search 標籤（related）方便日後 cross-reference
- 與 INBOX.md 不同：LINKS.md 是結構化的 URL 收藏，INBOX.md 是未分類片段
